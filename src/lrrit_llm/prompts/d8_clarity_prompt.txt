Dimension: The written report is clear and easy to read. The report is concise, written in easy to
understand, and inclusive language i.e., it is written to “inform rather than impress”.  (D8)

Core Definition:
The report is written clearly and accessibly, using inclusive language that supports understanding rather than demonstrating expertise.

Discriminators for the evidence level ratings:
GOOD Good evidence: Does the report make an active effort to be inclusive and accessible to a broad audience (including patients, families, and non-specialist staff), i.e., is there deliberate inclusivity rather than simply an absence of jargon?

SOME evidence: Can the report still be understood by non-specialist readers with reasonable effort even if inclusivity is not deliberate? 

LITTLE evidence: 
Is there little to no evidence, you should report this saying "I looked through the report and 
found no evidence of compassionate engagement".
